# Ascent Auto

**Ascent Auto** is a personal project that allows users to browse car listings, read short articles about each car, and submit reservation requests through a booking form.

## Technologies Used

- React
- React Router
- Tailwind CSS
- Git
- Create React App

## Features

- **Home Page** – main landing page with featured cars and information.
- **Car Articles** – individual pages for each car with descriptions and details.
- **Booking Form** – reservation form available with or without a pre-selected car.

## Future Plans

- Add a database for storing car listings and reservations.
- Implement a feature to change the landing page image on each visit randomly.

## Installation

To run the project locally:

```bash
git clone <repository-url>
cd ascent-auto
npm install
npm start
```

The app will be available at http://localhost:3000.
